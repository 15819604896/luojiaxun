# 线下选品中心 PRD

## 1. 项目概述
线下选品中心旨在为选品人员提供一个高效、直观的选品工作台，支持单品开发选品、Mood/Campaign选品以及已选品看板等功能，提升选品效率和准确性。

## 2. 需求背景及目的
**背景：**
目前，选品人在进行新一期选品时，缺乏对“历史已选款式”的全局可视化视图。选品决策主要依赖个人记忆或零散的表格记录，无法直观了解门店当前的款式结构风格线比例，选品时难以快速比对历史已选品数据，导致选款重复或同质化严重。

**目的：**
- **避免重复选款：** 选款时考虑店内已有款式，避免选太多重复款式，如避免选过多相近颜色或类似爆款衍生的款式。
- **结合目标选款：** 选款结合品类和风格线目标，确保各品类和风格线占比平衡，达成销售额目标。

## 3. 需求设计与说明
### 3.1 筛选区
- **数据范围：** 展示门店历史所有选品数据（排除库存（在途+在库+在店）为0的SKC）。
- **国际化需求：** 支持多语言切换，满足不同地区选品人员的使用需求。

### 3.2 核心功能模块
- **单品开发选品：** 支持对单个SKC进行选品决策（选中/取消），并设置目标门店和定级。
- **Mood/Campaign选品：** 提供可视化的图片瀑布流/网格视图，支持拖拽排序、批量定级，方便进行系列化选品。
- **已选品看板：** 提供全局视角的已选品统计和可视化图表，帮助选品人直观了解当前的款式结构和风格线比例。
